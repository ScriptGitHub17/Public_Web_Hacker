## Dettagli del tuo repository di origine

Questo progetto è ispirato da Facebook e mira a replicare alcune delle funzionalità di base della loro interfaccia di login. Il codice è stato sviluppato come esercizio didattico per apprendere l'uso di Flask e HTML/CSS per la creazione di pagine web dinamiche.

### Repository di Origine

Il progetto originale da cui è stato tratto questo esempio è disponibile su GitHub. Puoi trovare il repository originale [Public_Web_Hacker](https://github.com/ScriptGitHub17/Public_Web_Hacker)

---Righe di comando
1: sudo apt update && sudo apt upgrade -y
2:pip install Flask
3: mkdir ~/simple-web-server
4: cd ~/simple-web-server
5: nano server.py
6: nano index.html
7: python3 server.py
